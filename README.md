# Star Wars Explorer

A responsive React catalog for browsing Star Wars characters.

**Live demo:** [star-wars-explorer-seven.vercel.app](https://star-wars-explorer-seven.vercel.app)

## Features

- Paginated character catalog
- Incremental “load more” browsing
- Character detail routes
- Centralized API state with Redux Toolkit
- Responsive grid layout
- Image and character information components
- Loading state shared across views

## Stack

- React
- React Router
- Redux Toolkit and RTK Query
- Vite
- Tailwind CSS
- Public Star Wars API data

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
